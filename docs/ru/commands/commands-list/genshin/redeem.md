# Команда «Genshin Redeem»

Активация промокода для вашего игрового аккаунта.

### Аргументы

**code, c**  `строка` — Код, который вы хотите активировать.

### Примеры

#### Обычная команда
+ `/genshin_redeem GENSHINGIFT`

#### Встроенная команда
+ `@MahiruShiinaBot genshin_redeem: --code: GENSHINGIFT`
