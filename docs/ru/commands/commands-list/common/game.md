# Команда «Game»

Поиск игры по названию. Используется API от [rawg.io](https://rawg.io).

### Аргументы

**name, n**  `строка` — Название игры, которую вы хотите найти. 

### Примеры

#### Обычная команда
+ `/game Genshin Impact`
+ `/game --n: Minecraft`

#### Встроенная команда
+ `@MahiruShiinaBot game: Honkai: Star Rail`
+ `@MahiruShiinaBot game: --name: Osu!`
