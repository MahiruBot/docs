# Команда «Waifu»

Случайное изображение с аниме вайфу. Махиру использует API от [waifu.im](https://waifu.im) для данной команды.

### Аргументы

Аргументы для этой команды отсутствуют.

### Примеры

#### Обычная команда
+ `/waifu`

#### Встроенная команда
+ `@MahiruShiinaBot waifu`
