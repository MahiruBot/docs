# «Genshin Redeem» command

Redeem Genshin code on your game account.

### Arguments

**code, c**  `string` — The code you want to redeem.

### Examples

#### Default
+ `/genshin_redeem GENSHINGIFT`

#### Inline
+ `@MahiruShiinaBot genshin_redeem: --code: GENSHINGIFT`
