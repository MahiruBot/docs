# «Waifu» command

Get random waifu pic. Mahiru is using [waifu.im](https://waifu.im) API for this command.

### Arguments

This command does not have any arguments.

### Examples

#### Default
+ `/waifu`

#### Inline
+ `@MahiruShiinaBot waifu`
