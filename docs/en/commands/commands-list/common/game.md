# «Game» command

Search game by name. Search is performed using [rawg.io](https://rawg.io) API.

### Arguments

**name, n**  `string` — Name of the game you want to find. 

### Examples

#### Default
+ `/game Genshin Impact`
+ `/game --n: Minecraft`

#### Inline
+ `@MahiruShiinaBot game: Honkai: Star Rail`
+ `@MahiruShiinaBot game: --name: Osu!`
